Regenerate the executive summary report.

1. Activate the venv
2. Run `python scripts/generate_executive_summary.py`
3. Confirm the report was written to `reports/executive_summary.md`
4. List the generated chart assets in `reports/executive_summary_assets/`
5. Read the generated report and flag any metrics that look anomalous (e.g., negative percentages, implausible growth rates, missing data)
